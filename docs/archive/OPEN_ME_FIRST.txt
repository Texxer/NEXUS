❓ TROUBLESHOOTING:

Problem: "Rust not found"
→ Install from https://rustup.rs/

Problem: "Build fails"
→ Run: rustup update
→ Then: Build again

Problem: "Antivirus blocks cargo/build-script-build.exe"
→ This is a FALSE POSITIVE (100% safe)
→ Whitelist these folders in your antivirus:
   • C:\Users\YourUsername\.rustup\
   • C:\Users\YourUsername\.cargo\bin\
→ Reason: Rust & Cargo are legitimate tools, sometimes flagged
→ Then: Retry build

Problem: "IDE won't start"
→ See: NEXUS_WINDOWS_SETUP.md → Troubleshooting

Problem: "Shortcuts not working"
→ Edit: config/settings.json
→ Change: keybindings scheme

⚠️  IMPORTANT: Do NOT disable your antivirus!
   Just whitelist the Rust installation folder.
