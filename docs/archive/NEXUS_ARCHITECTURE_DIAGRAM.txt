## 🔄 Workflow After Build

```
IDE Built ✅
     │
     ├─── OPTION A: Use as-is
     │    └─ Run: IDE/dist/nexus-ide.exe
     │    └─ Or: ID/dist/RUN_NEXUS_IDE.bat
     │
     ├─── OPTION B: Install to Program Files
     │    ├─ Run: .\BUILD_NEXUS.ps1 -Action install
     │    └─ Then: nexus-ide.exe (from anywhere)
     │
     ├─── OPTION C: Share with Team
     │    ├─ Create ZIP: IDE/dist → nexus-ide.zip
     │    ├─ Share: nexus-ide.zip
     │    └─ Others: Extract & run RUN_NEXUS_IDE.bat
     │
     └─── OPTION D: Use on USB
          ├─ Copy: IDE/dist → USB drive
          ├─ Use: On any Windows machine
          └─ Run: RUN_NEXUS_IDE.bat (USB-portable!)
```
